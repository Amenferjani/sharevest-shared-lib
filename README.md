# sharevest-shared-lib

A TypeScript shared library providing TypeORM entities and NestJS DTOs for the ShareVest application ecosystem.

## Overview

`sharevest-shared-lib` is a centralized package designed to maintain consistency across ShareVest microservices by providing shared data models, database entities, and data transfer objects (DTOs) that can be reused across multiple applications.

## Features

- 📦 **TypeORM Entities** - Pre-configured database entities for common ShareVest models
- 🔄 **NestJS DTOs** - Standardized Data Transfer Objects for API requests and responses
- 📝 **Type-Safe** - Full TypeScript support with strict typing
- 🔌 **Easy Integration** - Simple to install and use across multiple projects
- 🎯 **Consistent API** - Ensure data consistency across all services
